# Camagru

## Configuration

Change this

cd Camagru/config

$DB_USER = "YOUR_ID";
$DB_PASSWORD = "YOUR_PWD";

## Install

php setup.php Create

## Uninstall

cd Camagru/config
php setup.php Drop
rm -rf Camagru/
